# Nova Framework - SDK - AdMob

> 包名：`com.solotopia.nova.framework.sdk.admob`
> 当前版本：`0.0.1`

Google AdMob 广告渠道插件，提供广告展示服务

## 安装

通过 Nova 私域 UPM 注册表以 UPM 依赖形式接入（注册表地址向 Nova Framework 内部开发人员索取）：

```json
"dependencies": {
  "com.solotopia.nova.framework.sdk.admob": "0.0.1"
}
```
