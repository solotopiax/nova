# Nova Framework - SDK - TGA

> 包名：`com.solotopia.nova.framework.sdk.tga`
> 当前版本：`0.0.15`

TGA 数据埋点插件，提供事件追踪服务

## 安装

通过 Nova 私域 UPM 注册表以 UPM 依赖形式接入（注册表地址向 Nova Framework 内部开发人员索取）：

```json
"dependencies": {
  "com.solotopia.nova.framework.sdk.tga": "0.0.7"
}
```
