# AdPlugin

> 包名：`com.solotopia.nova.framework.sdk.ad`
> 当前版本：`1.0.10`

广告聚合插件基类，支持 RV/Inter/Banner/AppOpen/InGameDisplay，内置打点

## 安装

通过 Nova 私域 UPM 注册表以 UPM 依赖形式接入（注册表地址向 Nova Framework 内部开发人员索取）：

```json
"dependencies": {
  "com.solotopia.nova.framework.sdk.ad": "1.0.10"
}
```
