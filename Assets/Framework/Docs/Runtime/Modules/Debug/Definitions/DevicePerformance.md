# DevicePerformance（已移除）

`DevicePerformance` 已不在当前源码中。

旧版 `Debug` 模块里曾包含设备性能评级与画质调节逻辑，但当前运行时实现已经移除这部分内容。当前 `DebugComponentInspector` 也不再绘制“硬件性能分级”面板。

如果你在旧资料里看到设备性能评级相关说明，应将其视为历史方案，而不是当前可依赖的 Runtime 能力。
