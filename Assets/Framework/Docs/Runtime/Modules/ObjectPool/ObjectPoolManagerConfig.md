# ObjectPoolManagerConfig

**类签名**：`public class ObjectPoolManagerConfig`
**命名空间**：`NovaFramework.Runtime`

对象池管理器的配置类。作为 `IObjectPoolManager.Initialize()` 的参数传入，用于在管理器初始化时提供配置信息。当前为空类，作为扩展预留点，后续可添加全局默认容量、过期时间等配置字段。
