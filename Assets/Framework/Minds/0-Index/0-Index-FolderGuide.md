# Nova Minds 文件夹指南

## 顶层目录

| 路径 | 用途 | 默认是否查询 |
|---|---|---|
| `0-Index/` | 入口与导航 | 是 |
| `1-Inbox/` | 草稿暂存 | 否 |
| `2-Areas/` | 当前有效知识 | 是 |
| `3-Resources/` | 外部资料与参考 | 按需 |
| `4-Archives/` | 旧决策、旧术语、旧记录 | 追溯时 |
| `Canvas/` | 辅助图示 | 默认否 |
| `.obsidian/` | 本地工作区状态 | 否 |

## `2-Areas/` 内部分区

| 路径 | 用途 |
|---|---|
| `ADR/` | 架构决策 |
| `Patterns/` | 长期模式与反模式 |
| `Glossary/` | 术语定义 |
| `MOC/` | 模块图谱 |
| `Snapshots/` | 审计快照 |

## 写入原则

- 当前实现说明写 `Docs`
- 长期结论写 `2-Areas`
- 草稿先放 `1-Inbox`
- 失效内容移入 `4-Archives`

## 默认忽略

- 本地工作区文件
- 工具过程文件
- 只对某一代协作栈成立的历史材料
