# 协作体系历史归档

这里存放 **旧协作栈** 的历史材料，包括：

- Claude / obs / hook / memory / agent 相关决策
- 旧的调度与审查流程
- 旧的知识库治理与命中机制
- 已不再作为 Codex 日常默认入口的协作模式

这些内容保留的目的只有两个：

1. 追溯历史演化
2. 在需要复盘旧方案时提供证据

它们 **不是** 当前 Nova 本体知识基线，也 **不是** Codex 日常开发默认读取入口。

## 结构

- `ADR/`：旧协作架构与工具布局决策
- `Patterns/`：旧协作模式、知识库治理模式、hook/obs 联动模式

## 使用规则

- 默认不要进入这里。
- 只有在复盘旧协作体系、迁移历史方案、解释为什么当年这么设计时才进入。
- 如果这里的内容与当前 `Docs`、源码或 `0-Index` 导航冲突，以当前事实层和现行导航为准。
